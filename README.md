# Bootcamp Microsoft AZ-204 - Projetos

Este repositório reúne os projetos desenvolvidos durante o **Bootcamp Microsoft AZ-204**, realizado na plataforma **DIO (Digital Innovation One)**.
O objetivo do bootcamp é explorar na prática os principais serviços da Azure voltados para desenvolvimento de aplicações em nuvem e arquitetura serverless.

Durante o programa são abordados conceitos importantes como:

* Desenvolvimento de aplicações cloud
* Microsserviços
* Azure Functions
* Armazenamento na Azure
* APIs e mensageria
* Monitoramento e escalabilidade

Cada projeto presente neste repositório representa um laboratório ou desafio proposto ao longo da trilha.

# Projetos

## 1️⃣ Microsserviço Serverless para Validação de CPF

Implementação de uma **Azure Function em Python** capaz de validar números de CPF através de uma API HTTP.

Tecnologias utilizadas:

* Python
* Azure Functions
* Serverless Architecture
* API HTTP

---

## 2️⃣ Deploy de API na Azure

Projeto focado em publicar e disponibilizar uma API na nuvem utilizando os serviços da Azure.

Principais conceitos:

* Deploy em cloud
* Gerenciamento de APIs
* Escalabilidade

---

## 3️⃣ Gerenciador de Catálogo (Estilo Netflix)

Projeto de backend serverless para gerenciamento de catálogo de conteúdos, inspirado em plataformas de streaming.

Conceitos aplicados:

* Azure Functions
* Banco de dados na nuvem
* APIs REST
