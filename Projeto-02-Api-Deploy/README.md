# Deploy de API na Azure

Este projeto demonstra o processo de publicação e gerenciamento de uma API utilizando os serviços da Microsoft Azure.

O laboratório foi realizado durante o Bootcamp AZ-204 da DIO.

## Objetivo

Publicar uma API na nuvem e gerenciar seu acesso utilizando o Azure API Management.

## Serviços utilizados

- Azure App Service
- Azure API Management
- Azure Portal

## Passos realizados

1. Criação do recurso no Azure
2. Deploy da API
3. Configuração do API Management
4. Teste da API via endpoint público

## Exemplo de chamada

GET /api/test

Resposta esperada:
{
"message": "API funcionando"
}


## Resultado

A API foi publicada com sucesso e pode ser acessada por meio de endpoint público gerenciado pelo Azure API Management.
